模块

在es6语法中，对于使用模块化编程，只需要三步

  • 导出
  • 导入
  • webpack编译
导出
  • export let a = 2
  • export {a,b,c}
  • export function show(){}
  • export class Person{}
导入
  • import * as mod1 from ‘./index.js’
  • import mod1 from ‘./index.js’
  • import ‘./index.js’
  • import {a,b as name2} from ‘./index.js’
  • let p = import(‘./index.js’)
