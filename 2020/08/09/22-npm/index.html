常用命令

初始化
  • npm init
    • npm init -y 可以跳过向导，快速生成
安装

  • npm install ( npm i )

    • 一次性把dependencies里面的依赖项全部安装

  • npm install 包名 ( npm i 包名 )

    • 只是下载

  • npm –save 包名 ( npm i -S 包名 )

    • 下载并保存依赖项( package.json里面的dependencies )
删除

  • npm uninstall 包名 ( nmp un 包名 )

    • 只删除，有依赖项信息依旧保存在package.json中

  • npm uninstall –save 包名 ( npm un -S 包名 )

    • 删除，并且删除依赖信息
帮助
  • npm –help
    • 帮助命令

解决 npm 下载慢/被墙问题

  • 因为npm服务器设置在国外，所以就注定存在 下载慢和被墙的问题
  • http://npm.taobao.org/ 淘宝的开发团队把npm在国内做了备份
安装淘宝的 cnpm

接下来把你的npm换成cnpm就可以从淘宝npm下载了

不安装cnpm仅仅想从淘宝镜像下载
  • 但是每次安装包都很麻烦，那我们就把这个选项加入配置文件中
